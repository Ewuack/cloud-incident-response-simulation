Containment Actions

1. Isolate the EC2 Instance
   - Remove from Auto Scaling Group
   - Detach from load balancer
   - Apply restrictive security group

2. Revoke Credentials
   - Rotate IAM role credentials
   - Invalidate temporary tokens

3. Preserve Evidence
   - Snapshot EBS volume
   - Export CloudTrail logs
   - Save GuardDuty findings

4. Block Malicious IPs
   - Update NACLs
   - Update security groups
